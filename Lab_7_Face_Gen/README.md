# Synthetic-Media
В данной лабораторной работе было выполненно 3 задачи:
1. Генерация лиц
2. Интерполяция лиц
3. Замена лиц

Для выполнения вычислений использовализь пердобученная модель модель stylegan2-ada pytorch/pretrained_models/e4e_ffhq_encode.pt
